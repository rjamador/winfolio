# Winfolio

A personal portfolio dressed up as a Windows 95 desktop — draggable windows, a
Start menu, a taskbar clock, and a few nostalgic touches.

🔗 **Live:** [ramador.dev](https://www.ramador.dev)

## What's inside

- Draggable, resizable windows with a working taskbar and Start menu
- Projects pulled live from the GitHub API
- Theming: pick a desktop color and text size (saved in your browser)
- English / Spanish, with browser-language detection
- Keyboard accessible and responsive down to mobile

## Stack

React · TypeScript · Vite · Tailwind CSS · TanStack Query · Zod · react-rnd

## Running locally

```bash
pnpm install
pnpm dev
```

| Command | What it does |
| --- | --- |
| `pnpm dev` | Start the dev server |
| `pnpm build` | Type-check and build for production |
| `pnpm test` | Run the test suite |
| `pnpm lint` | Lint the codebase |
